# Technology Stack - PMS TS

## Language
- **TypeScript**: Used throughout the project for type safety and modern JavaScript features.

## Frameworks and Libraries
- **React 18**: Core UI framework.
- **React Router DOM v6**: For client-side routing.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Flowbite & Flowbite React**: UI component library based on Tailwind CSS.
- **Axios**: HTTP client for API communication.
- **Moment.js & React Moment**: For date and time manipulation and formatting.
- **React Icons**: Icon library (Font Awesome, Ionicons, etc.).
- **Google Generative AI (Gemini)**: Integration for AI features.
- **JWT Decode**: For decoding JSON Web Tokens.
- **MinIO SDK**: For interacting with S3-compatible object storage.
- **Html2canvas**: For capturing screenshots/generating images from DOM elements.

## Build and Development Tools
- **Node.js**: Runtime environment.
- **Webpack**: Module bundler (via React App Rewired).
- **React App Rewired**: Used to customize Webpack configurations without ejecting.
- **Babel**: JavaScript compiler.
- **PostCSS**: For processing CSS with Tailwind.
- **Prettier**: Code formatting with Tailwind CSS plugin.
- **TypeScript Compiler (TSC)**: For transpiling TypeScript to JavaScript.
